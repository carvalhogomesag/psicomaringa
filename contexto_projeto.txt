# Contexto do Projeto: Psico Maringá

## Descrição Geral
O projeto "Psico Maringá" é uma landing page de alta conversão para serviços de psicologia na cidade de Maringá, PR. O objetivo principal é a captura de leads qualificados através de um formulário centralizado, sem a exposição de endereço físico ou contato telefônico direto.

## Tecnologias Utilizadas
- **Framework:** React 18+ com Vite.
- **Linguagem:** TypeScript.
- **Estilização:** Tailwind CSS (utilizando a nova sintaxe @import "tailwindcss").
- **Animações:** Motion (motion/react).
- **Ícones:** Lucide React.
- **Roteamento:** React Router DOM v6.

## Estrutura de Conteúdo e SEO
- **Página Inicial (Home):** Apresentação dos serviços, especialidades, depoimentos, FAQ e prévia do blog.
- **Especialidades:** Páginas dinâmicas para cada tipo de atendimento (Ansiedade, Depressão, Terapia de Casal, Infantil, etc.).
- **Blog:** Sistema de artigos informativos focados em SEO local, destacando a importância de profissionais qualificados.
- **FAQ:** Seção de dúvidas frequentes oculta por padrão (toggle) para manter o design limpo, mas acessível para SEO.
- **Localização:** Foco em bairros de Maringá (Centro, Zona 07, Vila Operária, etc.) citados estrategicamente no texto.

## Funcionalidades Principais
- **Formulário de Leads:** Componente centralizado (`LeadForm.tsx`) com validação e feedback de sucesso.
- **Mobile First:** Design otimizado para dispositivos móveis com botão flutuante de "Solicitar Contato".
- **Conversão:** Múltiplos pontos de CTA (Call to Action) espalhados pelo site (Hero, Especialidades, Testemunhos, Footer).
- **SEO Dinâmico:** Componente `SEO.tsx` para gerenciar títulos e descrições de cada página.

## Diretrizes de Design
- **Paleta de Cores:** Tons de esmeralda (`emerald`) e zinco (`zinc`) para transmitir profissionalismo e calma.
- **Tipografia:** Inter (sans-serif) para legibilidade.
- **Abordagem:** Ética e profissional, evitando dar conselhos diretos e focando na importância do acompanhamento especializado.

## Arquivos Chave
- `/src/constants.ts`: Contém todos os dados de especialidades, bairros, FAQs e posts do blog.
- `/src/components/LeadForm.tsx`: Lógica e interface do formulário de captura.
- `/src/components/Layout.tsx`: Estrutura global (Header, Footer e Botão Flutuante).
- `/src/Home.tsx`: Composição da landing page principal.
